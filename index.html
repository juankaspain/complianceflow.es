Una plataforma, cuatro bloques de APIs

ComplianceFlow está organizado en bloques modulares: Facturación (SII/Verifactu), Identidad (KYC), Riesgo (fraude) y Documentos (OCR + contratos). Integra solo lo que necesitas y escala al resto cuando tu negocio crezca.

/sii · /verifactu

Facturación y AEAT

Convierte facturas en envíos SII válidos para la AEAT y genera tickets Verifactu sin reescribir tu ERP ni tu software de facturación.

  • Mapeo automático de campos de factura.
  • Gestión de errores y reintentos.
  • Listo para futuras obligaciones de e‑factura.

/kyc

Onboarding digital

Procesa DNI/pasaporte + selfie, valida identidad y calcula un score de riesgo compatible con PSD2 y políticas internas.

  • Detección de documentos manipulados.
  • Comparación biométrica selfie vs documento.
  • Respuestas explicables para compliance.

/score/risk

Fraude transaccional

Calcula en milisegundos el riesgo de cada pago o movimiento combinando reglas deterministas e IA.

  • Reglas configurables por producto.
  • Modelos entrenados con patrones de fraude.
  • Salida 0‑100 con motivos del bloqueo.

/extract · /summarize

Documentos y contratos

Extrae datos clave de facturas, DNIs, contratos y expedientes, y genera resúmenes ejecutivos para decisiones rápidas.

  • OCR multiformato (PDF, imagen).
  • Detección de cláusulas sensibles.
  • Resúmenes listos para informes internos.

Cómo funciona ComplianceFlow

El flujo es siempre el mismo: ingesta por API, extracción y normalización, aplicación de reglas + IA y respuesta JSON lista para guardar en tu sistema.

  1. 1. Ingesta

    Envías documentos o eventos a través de HTTPS con tu API key.

  2. 2. Normalización

    Se limpian, extraen y validan los datos con modelos especializados.

  3. 3. Reglas + IA

    Se aplican reglas normativas e IA entrenada en patrones de riesgo.

  4. 4. Respuesta y logs

    Recibes una respuesta JSON con identificadores para auditoría.

Preguntas frecuentes

        


        

          

            ¿Necesito cambiar mi ERP o core bancario para usar ComplianceFlow?
            

              
No. Normalmente se añade una capa ligera que transforma tus datos internos
               al formato JSON que espera cada endpoint, manteniendo tus sistemas actuales.

            

          

          

            ¿Dónde se alojan los datos?
            

              
El procesamiento se realiza en centros de datos de la UE sobre Azure;
               los detalles concretos de retención se pactan en el contrato de servicio.

            

          

          

            ¿Cumple con GDPR y requisitos de supervisores?
            

              
ComplianceFlow se diseña siguiendo principios de privacy‑by‑design,
               cifrado en tránsito y en reposo y registro de actividad para auditoría.
