# ComplianceFlow Landing

Landing estática para **ComplianceFlow**, una plataforma de APIs de compliance
(SII, Verifactu, PSD2, KYC, fraude y análisis documental).

- HTML5 semántico y accesible.
- CSS responsive optimizado para Core Web Vitals.
- JS ligero para demos conectadas a `api.complianceflow.es` (sandbox).
